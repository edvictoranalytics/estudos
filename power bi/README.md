O que é um DASHBOARD?

Os dashboards permitem que os consumidores de relatório criem um artefato de dados direcionados que é personalizado apenas para eles.

Características
- Visuais Fixados
- Visuais de relatórios distinhos

Como eu crio um Dasgbaord?
No power bi serviço.

Qual a diferença entre um relatório e um Dashboard?
- Vários conjuntos de dadosou reports
- Não possuem painéis Filtro, visualização e Campos (Dashboards)
- Podem ter apenas uma página (Dashboards) | Relatório (Várias páginas)
- Quem tem acesso ao dashboard não tem acesso aos dados "diretamente" | Quem tem acesso ao relatório tem acesso aos dados
- Ambos podem ser atualizados
- Manipulação dos blocos
