O que é um DASHBOARD?

Os dashboards permitem que os consumidores de relatório criem um artefato de dados direcionados que é personalizado apenas para eles.

Quando usar um Dashboard?

Os dashboards devem ser uma coleção de várias fontes, não apenas uma "plataforma de lançamento" para relatórios.

Características
- Visuais Fixados
- Visuais de relatórios distinhos

Como eu crio um Dasgbaord?
No power bi serviço.

Qual a diferença entre um relatório e um Dashboard?
- Vários conjuntos de dadosou reports
- Não possuem painéis Filtro, visualização e Campos (Dashboards)
- Podem ter apenas uma página (Dashboards) | Relatório (Várias páginas)
- Quem tem acesso ao dashboard não tem acesso aos dados "diretamente" | Quem tem acesso ao relatório tem acesso aos dados
- Ambos podem ser atualizados
- Manipulação dos blocos

Alertas em Dashboards
- Notificalção de usuário
- Disponível no Power BI Service

Presente em Elementos:
- Medidores, cartões e KPIs
- Configuração ex: 24h

Live Page - Relatório
- Garantindo dinâmica dos dados
- Atualização das infos
- Relatório dinâmicos
- Fixa o relatório no dashboard

Streaming no Power BI
- Pegando dados de fonte heterogêneas em tempo real
- Capacidade de exibir dados em tempo real
- Grande volume e velocidade
- Dashboards são telas de histórias
- Dados em cache temporário
