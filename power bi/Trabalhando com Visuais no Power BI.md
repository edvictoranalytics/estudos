# Formalizando conceitos do Power BI

<dataset> coleção de dados organizados usados para análises e modelagem de dados.

Os conjuntos de dados estruturados frequentemente organizam os pontos de dados em tabelas, com linhas e colunas definidas.

Os conjuntos de dados não estruturados podem conter vários formatos, como arquivos de texto, imagens e áudio.

Embora nem todos os conjuntos de dados envolvam dados estruturados, eles sempre têm alguma estrutura geral, sejam esquemas definidos ou sintaxe mal organizada em formatos de dados semiestruturados, como JSON ou XML.

Exemplos de datasets incluem:

Datasets de atendimento ao cliente que acompanham as interações e as resoluções do suporte.
Datasets de fabricação que monitoram métricas de desempenho de equipamentos.
Datasets de vendas que analisam padrões de transações e comportamento dos consumidores.
Datasets de marketing que medem a eficácia e o engajamento das campanhas.
As organizações frequentemente usam e mantêm vários datasets para apoiar várias iniciativas de negócios, incluindo análise de dados e business intelligence (BI).

Big data, em particular, depende de dataset complexos e maciços para proporcionar valor. Quando devidamente coletados, gerenciados e analisados usando análise de dados de big data, esses datasets podem ajudar a descobrir novos insights e permitir a tomada de decisões baseada em dados.
